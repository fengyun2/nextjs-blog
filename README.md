# NextBlog
> 基于 nextjs + mdx 实现的博客系统

## 特性

- 基于 nextjs + mdx 实现
- 基于 nextjs + tailwindcss 实现

## 启动

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

然后在浏览器打开 [http://localhost:3000](http://localhost:3000) 查看结果.
